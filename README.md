# Fishing Macro

Auto-fishing macro for Minecraft Skyblock with Kill Aura functionality.

## Features

✅ **Auto Fishing** - Automatically clicks right mouse button when fishing rod is active  
✅ **Kill Aura** - Automatically attacks nearby hostile mobs  
✅ **Keybinding Controls** - Toggle on/off with keyboard shortcuts  
✅ **Configurable Range** - Adjust detection range for mobs

## Installation

1. Download the latest `.jar` from Releases
2. Place it in your `.minecraft/mods` folder
3. Launch Minecraft with Fabric Loader

## Controls

- **F** - Toggle Auto Fishing
- **K** - Toggle Kill Aura

## Building

```bash
./gradlew build
```

The compiled JAR will be in `build/libs/fishingmacro-1.0.0.jar`
